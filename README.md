# 量化策略项目（Quant Strategy Project）

## 📌 项目简介
本项目实现了多种经典量化交易策略，并对策略进行回测与绩效评估。

## 🧠 策略列表
- 均线策略（Moving Average Strategy）
- 布林带策略（Bollinger Bands Strategy）

## 📊 功能模块
- 策略信号生成（strategy）
- 收益曲线计算（cumulative return）
- 策略绩效分析（performance）

## 🛠 技术栈
- Python
- pandas
- numpy
